# Huffman Compression

## Description
This project implements a file compression and decompression system using Huffman Coding.

## Features
- Compress and decompress files
- Supports large files
- Custom buffer sizes
- Uses Min Heap and Linked List

## Technologies
- C++
- Data Structures (Heap, Linked List, Tree)

## How it works
The program builds a Huffman Tree based on character frequencies, assigns binary codes, and compresses the file accordingly.

## How to run
1. Compile the project
2. Run the program
3. Choose compress or decompress
